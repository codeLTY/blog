# flex

  • display:none
    • 不会占据位置, 原本的位置也会在页面布局中消失
  • visibility:hidden
    • 该元素会隐藏, 但原本的位置不会在页面布局中消失
    • 该元素身上如果有绑定事件也不会被触发
  • opacity: 0
    • 只是把该元素的透明度设置为不可见,元素会隐藏,但原本的位置不会在页面布局中消失
    • 该元素身上如果绑定事件点击会被触发

# transition、animation、transform的区别

  • transition 只适用于单一的动画,其实质就是从一种效果逐渐变为另一种,需要通过事件来触发
  • animation 用于复杂的动画,自己触发
  • transform 用来改变元素的旋转、缩放、移动、倾斜等

# CSS优先级

  • 不同级别：!important > 行内样式 > ID选择器 > 类选择器 > 标签 > 通配符 > 继承 > 浏览器默认属性
  • 同一级别: 同一级别中后写的会覆盖先写的样式
