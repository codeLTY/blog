# 圣杯布局

  • 首先给定元素height,然后全都左浮动
  • 左右两个元素固定宽度,中间元素宽度100%
  • 使用margin-left: -100%; 把左边元素拉回到左边
  • 使用margin-left: -300px(ps:此处为右边元素的宽度); 把右边元素拉回到右边
  • 此时需要把左右元素放到两边,就需要让两边有边距,所以先给父元素padding: 0 300px(ps:此处需要根据左右元素宽度来决定),然后两边已经产生边距，不过两边元素还是在中间元素上面显示;
  • 最后就需要把两边元素固定在左右两侧,使用position: relative; 左边元素 left: -300px; 右边元素 right: -300px;
* {
    margin: 0;
    padding: 0;
}
.container {
    padding: 0 300px;
}
.center {
    float: left;
    width: 100%;
    min-height: 300px;
    background: pink;
}

.left {
    float: left;
    width: 300px;
    min-height: 300px;
    margin-left: -100%;
    position: relative;
    left: -300px;
    background: lightblue;
}

.right {
    float: left;
    width: 300px;
    min-height: 300px;
    margin-left: -300px;
    position: relative;
    right: -300px;
    background: yellowgreen;
}

<div class='container'>
    <div class='center'>
        <h1>圣杯布局</h1>
    </div>
    <div class='left'></div>
    <div class='right'></div>
</div>

# 双飞翼布局

  • 双飞翼布局相较于圣杯布局更加简洁,只是中间元素多加了一层div而已
* {
    margin: 0;
    padding: 0;
}

.center-container {
    float: left;
    width: 100%;
    min-height: 300px;
    background: pink;
}

.center {
    margin: 0 300px;
}

.left {
    float: left;
    width: 300px;
    min-height: 300px;
    margin-left: -100%;
    background: lightblue;
}

.right {
    float: left;
    width: 300px;
    min-height: 300px;
    margin-left: -300px;
    background: yellowgreen;
}

<div class='container'>
    <div class='center-container'>
        <div class="center">
            <h1>双飞翼布局</h1>
        </div>
    </div>
    <div class='left'></div>
    <div class='right'></div>
</div>

# flex布局

  • flex布局是最为简洁的,只需给外层的div设置display: flex;左右元素固定宽高,中间元素flex:1;给定高度即可
  • 唯一不足就是旧浏览器不兼容
* {
    margin: 0;
    padding: 0;
}

.container {
    display: flex;
}

.left {
    width: 300px;
    min-height: 300px;
    background: lightblue;
}

.center {
    min-height: 300px;
    flex: 1;
    background: pink;
}

.right {
    width: 300px;
    min-height: 300px;
    background: yellowgreen;
}

<div class='container'>
    <div class='left'></div>
    <div class="center">
        <h1>flex布局实现三栏布局</h1>
    </div>
    <div class='right'></div>
</div>
